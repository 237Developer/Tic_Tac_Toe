# Tic_Tac_Toe

A modern, clean, and interactive version of the classic Tic Tac Toe game, playable directly in the browser. 

![Project Overview](images/Pasted%20image.png)  

## Technologies Used
* **HTML**: Structuring the 3x3 game board and UI elements.
* **CSS**: Modern styling and layout.
* **JavaScript**: Core game logic and turn management. 

## How to Play
The rules of the game are simple: the player who first aligns three tokens wins the game. 

You can try it here: [Tic Tac Toe Game](https://237developer.github.io/Tic_Tac_Toe/)
